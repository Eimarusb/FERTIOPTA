FERTIOPTA

FERTIOPTA es un sistema automatizado de fertirriego orientado al cultivo de papa, diseñado para optimizar el suministro de agua y nutrientes mediante procesos de automatización y control hidráulico.

🎯

Objetivo Principal

Desarrollar un sistema automatizado de fertirriego para optimizar el cultivo de papa.

⚙️

Operación Automática

Implementar modos automáticos de operación hidráulica.

🛢️

Control del Sistema

Automatizar el control del tanque y las cargas del sistema.

💧

Eficiencia Hídrica

Optimizar el aprovechamiento del agua y nutrientes.

Componentes del Sistema

Arduino OPTA

El cerebro del sistema. Ejecuta la lógica de control.

Electrobomba

Genera el flujo hidráulico necesario.

Electroválvulas

Controlan automáticamente el paso del agua.

Tanque

Almacenamiento de agua.

Tuberías

Rutas hidráulicas de transporte.

Cuarto Semestre

Fase inicial de ideación, levantamiento de requerimientos y diseño estructural preliminar del esquema P&ID para el sistema de fertirriego.

Quinto Semestre

Montaje del prototipo a escala, cableado del controlador industrial Arduino OPTA y programación base de los algoritmos de conmutación hidráulica.

🔒 PRÓXIMAMENTE

Sexto Semestre

Pruebas de campo integradas, calibración de tiempos de dosificación de nutrientes, optimización del sistema web y documentación final.

Modos de Operación

Oxigenación

Recirculación para incrementar niveles de oxígeno disuelto.

Riego Regular

Hidratación convencional directa en las líneas de riego.

Fertirriego

Inyección automática de nutrientes en el caudal de agua.

Llenado Automático

Recuperación del volumen mínimo seguro del tanque.
