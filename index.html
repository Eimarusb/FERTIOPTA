FERTIOPTA

FERTIOPTA es un sistema automatizado de fertirriego orientado al cultivo de papa, diseñado para optimizar el suministro de agua y nutrientes mediante procesos de automatización y control hidráulico.


El proyecto busca mejorar la eficiencia del riego, reducir desperdicios de agua y facilitar la integración de tecnologías aplicadas al sector agrícola.

🎯

Objetivo Principal

Desarrollar un sistema automatizado de fertirriego para optimizar el cultivo de papa.

⚙️

Operación Automática

Implementar modos automáticos de operación hidráulica.

🛢️

Control del Sistema

Automatizar el control del tanque y las cargas del sistema.

💧

Eficiencia Hídrica

Optimizar el aprovechamiento del agua y nutrientes.

Componentes del Sistema

El sistema FERTIOPTA está compuesto por elementos encargados de controlar, transportar y distribuir agua y nutrientes durante la operación automática.

🧠

Arduino OPTA

El cerebro del sistema. Ejuta la lógica de control y coordina todos los procesos.

💧

Electrobomba

El corazón del sistema. Genera el flujo hidráulico necesario para operar.

⚙️

Electroválvulas

Las manos del sistema. Controlan automáticamente el paso del agua.

🛢️

Tanque

El centro de almacenamiento de agua para la operación.

🔩

Tuberías y Mangueras

Las rutas hidráulicas del sistema para transportar agua y nutrientes.

Funcionamiento Teórico

El sistema FERTIOPTA basa su lógica en una máquina de estados controlada por un autómata programable. A través del procesamiento de señales procedentes de la matriz de sensores, el controlador discrimina de manera automática el escenario óptimo de trabajo. La conmutación entre los diferentes modos operativos garantiza tanto el cuidado metabólico del cultivo de papa como la integridad electromecánica de la planta hidráulica.

🌱

Riego Normal

Modo base del sistema. Activa la electrobomba de acoplamiento y abre la electroválvula de salida para transportar agua limpia de manera directa a los canales de distribución, hidratando el cultivo de forma convencional.

🧪

Fertirriego

Etapa de nutrición asistida. Durante esta fase, el sistema dosifica compuestos químicos disueltos y los inyecta dinámicamente en el flujo de salida, permitiendo que la planta absorba agua y macroelementos de forma simultánea.

🌀

Oxigenación

Rutina de mantenimiento cíclico. Para evitar la sedimentación de nutrientes y la proliferación anaeróbica, el controlador desvía el caudal a través de la electroválvula de oxigenación, provocando una recirculación interna cerrada dentro del tanque principal.

🚨

Llenado Prioritario

Algoritmo de protección crítica. Si los sensores detectan un nivel de fluido inferior al umbral mínimo, el sistema interrumpe cualquier proceso en curso para abrir la electroválvula de llenado, reabasteciendo el tanque y previniendo la cavitación o marcha en seco de la bomba.
